Information We Collect

Permissions

The permissions requested by our extension are solely for the purpose of providing the functionality described:

  • activeTab: Allows the extension to interact with the currently active tab to insert generated data into form fields.
  • storage: Used to save user preferences and settings to enhance the user experience.
  • contextMenus: Enables the creation of custom context menu items for easy access to the extension's features.
  • scripting: Required to execute scripts that insert the generated data into web pages.
